# 图片资源说明

## 需要准备的图片

### 1. avatar-placeholder.png
- **用途**: 用户头像占位图
- **尺寸**: 200x200 像素
- **格式**: PNG（支持透明背景）
- **位置**: `templates/official/assets/images/avatar-placeholder.png`

你可以：
1. 使用在线工具生成一个简单的头像占位图
2. 使用纯色背景 + 图标的方式
3. 从免费图片网站下载默认头像图片

### 推荐设计
- 灰色圆形背景
- 中间白色的人像图标
- 或者使用字母头像生成器

### 在线工具推荐
- https://ui-avatars.com/ - 字母头像生成器
- https://avatar.oxro.io/ - 随机头像生成器
- Canva - 自己设计简单头像

## 示例URL（临时使用）
如果暂时没有准备图片，可以使用：
```
https://ui-avatars.com/api/?name=User&size=200&background=6c757d&color=fff
```
