Carefree ConfigTagService Multisite Test

Using ThinkPHP Variables (from $templateData)

Site Name (Think): {$site.site_name}

Site Code (Think): {$site.site_code}

Logo (Think): {$site.logo}

Using Carefree Config Tags (from ConfigTagService)

Site Name (Carefree): {carefree:config name="site_name" default="Default Site" /}

Site Code (Carefree): {carefree:config name="site_code" default="default" /}

Logo (Carefree): {carefree:config name="logo" default="/default-logo.png" /}

Nested Config Access

SEO Keywords (nested dot notation): {carefree:config name="seo.keywords" default="default keywords" /}

SEO Keywords (flat notation): {carefree:config name="seo_keywords" default="default keywords" /}

SEO Title (nested): {carefree:config name="seo.title" default="Default Title" /}

Site URL Fields (前端网站网址)

Site URL (前端网站网址): {carefree:config name="site_url" default="http://localhost" /}

URL (别名，优先site_url): {carefree:config name="url" default="http://localhost" /}

Domain (域名，兼容): {carefree:config name="domain" default="localhost" /}

Template package code: {$package.code}

Config keys available: {$config|json_encode}

使用说明

  • site_url: 前端网站网址（推荐使用）
  • url: site_url的别名，如果site_url为空则使用domain
  • domain: 域名字段，保留用于兼容旧模板
