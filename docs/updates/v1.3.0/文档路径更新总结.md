## 更新内容详情

### 路径替换规则

所有文档中的路径引用按以下规则进行了替换：

1. **目录路径**
   - `api/` → `backend/`
   - `backend/` → `frontend/`

2. **命令中的目录**
   - `cd api` → `cd backend`
   - `cd backend` → `cd frontend`

3. **配置文件路径示例**
   - `api/config/database.php` → `backend/config/database.php`
   - `api/public/uploads` → `backend/public/uploads`
   - `api/html/` → `backend/html/`
   - `backend/src/utils/request.js` → `frontend/src/utils/request.js`
   - `backend/dist` → `frontend/dist`

4. **Nginx配置示例**
   ```nginx
   # 更新前
   root /var/www/carefreecms/api/public;
   root /var/www/carefreecms/backend/dist;

   # 更新后
   root /var/www/carefreecms/backend/public;
   root /var/www/carefreecms/frontend/dist;
   ```
