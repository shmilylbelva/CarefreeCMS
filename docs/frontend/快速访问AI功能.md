### 🚀 快速开始


打开浏览器访问：`http://localhost:5173`

使用管理员账号登录：

#### 4. 查找AI菜单

### 📖 功能说明

#### AI配置管理 (`/ai-configs`)
- 添加AI服务配置（OpenAI、Claude、DeepSeek等）
- 设置API密钥和模型参数
- 测试AI连接
- 管理多个AI配置

#### 批量生成任务 (`/ai-tasks`)
- 创建文章批量生成任务
- 查看任务进度和统计
- 管理生成的文章
- 实时查看生成记录

### 🎯 首次使用流程

1. **点击"AI文章生成" → "AI配置管理"**
2. **点击"添加配置"按钮**
3. **填写配置信息**：
   ```
   配置名称: OpenAI GPT-3.5
   AI提供商: openai
   API密钥: sk-your-api-key
   模型名称: gpt-3.5-turbo
   最大Tokens: 2000
   温度参数: 0.7
   ```
4. **点击"测试"按钮**验证配置
5. **点击"AI文章生成" → "批量生成任务"**
6. **点击"创建任务"按钮**
7. **填写任务信息**：
   ```
   任务名称: 测试批量生成
   生成主题: 人工智能应用,机器学习入门
   AI配置: 选择刚创建的配置
   目标分类: 选择一个分类
   生成数量: 2
   ```
8. **点击"确定"创建任务**
9. **点击"启动"按钮**开始生成
10. **观察进度条**，等待生成完成
11. **点击"查看记录"**查看生成的文章

### 💡 提示

- 第一次使用建议先生成1-2篇测试效果
- 确保后端服务（`php think run`）正在运行
- 确保API密钥有效且有足够余额
- 生成过程中页面会自动刷新进度

### ❓ 如果看不到菜单

1. **清除浏览器缓存**并刷新页面（Ctrl+F5）
2. **检查是否已登录**
3. **确认前端服务正在运行**
4. **检查浏览器控制台**是否有错误
