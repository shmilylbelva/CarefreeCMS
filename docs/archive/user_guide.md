## 登录系统

1. 访问管理后台地址
2. 输入用户名和密码
3. 点击"登录"按钮

> 默认账号：admin / admin123（首次登录请修改密码）

## 仪表板

登录成功后，您将看到仪表板页面，显示：

- **系统统计**: 文章数、分类数、标签数、用户数
- **最新文章**: 最近更新的文章列表
- **快捷功能**: 常用功能的快捷入口

## 内容管理

### 文章管理

#### 创建文章

1. 点击左侧菜单"内容管理" → "文章列表"
2. 点击右上角"创建文章"按钮
3. 填写文章信息：
   - **标题**: 文章标题（必填）
   - **分类**: 选择文章分类（必填）
   - **标签**: 选择或创建标签
   - **封面图片**: 上传文章封面（可选）
     - 建议尺寸：800x450像素（16:9）
     - 支持格式：jpg、png、gif
     - 大小限制：2MB
   - **摘要**: 文章简介
   - **内容**: 使用富文本编辑器编写内容（必填）
   - **作者**: 显示的作者名称
   - **文章属性**: 置顶、推荐、热门等
   - **发布状态**: 草稿/已发布
   - **发布时间**: 自定义发布时间
   - **SEO信息**:
     - 标题：搜索引擎显示的标题（留空自动提取）
     - 关键词：页面关键词（留空自动提取）
     - 描述：页面描述（留空自动提取内容前200字）

4. 点击"保存草稿"或"发布文章"

#### 编辑文章

1. 在文章列表中找到要编辑的文章
2. 点击"编辑"按钮
3. 修改文章信息
4. 点击"更新文章"保存

#### 删除文章

1. 在文章列表中勾选要删除的文章
2. 点击"批量删除"按钮
3. 确认删除操作

### 分类管理

#### 创建分类

1. 点击"内容管理" → "分类管理"
2. 点击"添加分类"
3. 填写分类信息：
   - **名称**: 分类名称（必填）
   - **标识**: URL友好的标识符
   - **父分类**: 选择上级分类（可选）
   - **排序**: 数字越大越靠前
   - **描述**: 分类描述
   - **模板**: 自定义模板文件
   - **状态**: 启用/禁用

4. 点击"保存"

#### 分类排序

- 直接拖拽分类行来调整顺序
- 或修改"排序"字段的数值

### 标签管理

#### 创建标签

1. 点击"内容管理" → "标签管理"
2. 点击"添加标签"
3. 填写标签信息：
   - **名称**: 标签名称（必填）
   - **标识**: URL标识符
   - **描述**: 标签描述
   - **状态**: 启用/禁用

4. 点击"保存"

### 文章属性

文章属性用于标记文章的特殊状态，如置顶、推荐、热门等。

#### 创建属性

1. 点击"内容管理" → "文章属性"
2. 点击"添加属性"
3. 填写属性信息：
   - **名称**: 属性名称
   - **标识**: 唯一标识符
   - **排序**: 显示顺序
   - **状态**: 启用/禁用

### 单页管理

单页用于创建独立的内容页面，如"关于我们"、"联系我们"等。

#### 创建单页

1. 点击"内容管理" → "单页管理"
2. 点击"创建单页"
3. 填写页面信息：
   - **标题**: 页面标题（必填）
   - **URL别名**: 页面访问路径（必填）
   - **封面图片**: 页面封面
   - **内容**: 页面内容（必填）
   - **模板**: 自定义模板
   - **SEO信息**: 标题、关键词、描述（自动提取）
   - **排序**: 显示顺序
   - **状态**: 已发布/草稿

4. 点击"保存"

## 媒体库

媒体库统一管理所有上传的文件。

### 上传文件

1. 点击"媒体库"
2. 点击"上传文件"按钮
3. 选择文件上传
   - 支持图片：jpg、jpeg、png、gif、webp、ico
   - 支持文档：pdf、doc、docx、xls、xlsx
   - 支持压缩包：zip、rar
   - 大小限制：10MB

### 文件筛选

- **按类型**: 选择"全部"、"图片"、"视频"、"文档"、"其他"
- **按名称**: 输入文件名搜索
- **按日期**: 选择日期范围查询

### 文件操作

- **复制链接**: 点击"复制链接"获取文件URL
- **删除文件**: 点击"删除"移除文件
- **批量删除**: 勾选多个文件后批量删除

## SEO管理

### 静态生成

将动态页面生成为静态HTML文件，提升访问速度和SEO效果。

#### 生成选项

1. 点击"SEO管理" → "静态生成"
2. 选择生成内容：
   - **生成首页**: 生成网站首页
   - **生成文章列表**: 生成所有文章列表页
   - **生成单篇文章**: 选择特定文章生成
   - **批量生成所有**: 一键生成所有页面

3. 点击相应的"生成"按钮
4. 等待生成完成
5. 查看生成日志

#### 生成日志

- 查看每次生成的详细信息
- 筛选生成类型和状态
- 批量删除历史日志
- 清空指定天数前的日志

### Sitemap生成

生成网站地图，帮助搜索引擎更好地索引网站内容。

1. 点击"SEO管理" → "Sitemap生成"
2. 配置Sitemap选项：
   - 包含文章
   - 包含分类
   - 包含单页
   - 包含标签

3. 点击"生成Sitemap"
4. 生成的文件位于网站根目录 `sitemap.xml`

#### 网站信息

- **图片最大高度**: 自动压缩超高图片

- **缩略图高度**: 缩略图尺寸

#### 第三方代码

- **PC端统计代码**: 添加百度统计、Google Analytics等代码

### 缓存管理

系统提供灵活的缓存管理功能，支持多种缓存驱动和便捷的缓存操作。

#### 缓存信息查看

1. 点击"系统管理" → "缓存管理"
2. 查看当前缓存统计信息：
   - 缓存类型和驱动
   - 已用内存/存储空间
   - 键数量/文件数量
   - 命中率等性能指标

#### 缓存驱动切换

系统支持两种缓存驱动：

**File驱动（文件缓存）**
- 默认驱动，无需额外配置
- 将缓存存储在文件系统中
- 适合中小型网站

**Redis驱动（内存缓存）**
- 高性能内存缓存
- 需要安装PHP Redis扩展和Redis服务器
- 适合高并发、大流量网站

**切换步骤：**

1. 在"缓存驱动设置"卡片中选择目标驱动
2. 如果选择Redis：
   - 填写服务器地址（默认：127.0.0.1）
   - 填写端口（默认：6379）
   - 填写密码（如有）
   - 点击"测试连接"验证配置
3. 点击"应用更改"
4. 确认操作（切换会清空现有缓存）
5. 等待页面自动刷新

**注意事项：**
- 切换缓存驱动会清空现有缓存
- Redis驱动需要先安装PHP Redis扩展
- 建议在测试环境验证配置后再在生产环境应用

#### 缓存操作

**清空所有缓存**
- 点击"清空所有缓存"卡片中的"立即清空"按钮
- 清除系统中的所有缓存数据

**清除模板缓存**
- 点击"清除模板缓存"卡片中的"清除模板"按钮
- 清除编译后的模板文件
- 适用于模板修改后需要重新编译的场景

**缓存预热**
- 点击"缓存预热"卡片中的"开始预热"按钮
- 预加载常用数据到缓存
- 提升系统响应速度

### 操作日志

记录所有用户的操作行为，便于审计和追踪。

#### 查看日志

1. 点击"系统管理" → "操作日志"
2. 可按以下条件筛选：
   - **用户名**: 操作用户
   - **模块**: 操作模块（文章、分类等）
   - **操作类型**: 创建、更新、删除等
   - **状态**: 成功/失败
   - **日期范围**: 指定时间段

#### 日志详情

点击"详情"按钮查看操作的完整信息：
- 操作用户
- 操作时间
- IP地址
- 用户代理
- 请求URL
- 请求参数
- 执行时间
- 错误信息（如有）

#### 日志管理

- **批量删除**: 勾选多条日志批量删除
- **清空日志**: 删除指定天数前的所有日志
  - 最少保留7天
  - 建议定期清理以节省空间

## 最佳实践

### 内容发布流程

1. 创建必要的分类和标签
2. 准备文章内容和配图
3. 创建文章并保存为草稿
4. 预览确认无误后发布
5. 生成静态页面
6. 前端查看效果

### SEO优化建议

1. **关键词**: 在标题、描述、内容中合理使用关键词
2. **图片优化**:
   - 使用合适的图片尺寸
   - 添加图片alt属性
   - 压缩图片大小
3. **内容质量**:
   - 原创内容为主
   - 定期更新
   - 内容结构清晰
4. **静态化**: 定期生成静态页面提升访问速度
5. **Sitemap**: 每次更新内容后重新生成sitemap

### 安全建议

1. 定期修改密码
2. 不同用户分配合适的角色权限
3. 定期查看操作日志
4. 定期备份数据库和文件
5. 及时更新系统版本

## 常见问题

### 1. 上传文件失败

- 检查文件大小是否超限
- 确认文件格式是否支持
- 检查上传目录权限

### 2. 静态页面无法访问

- 检查Web服务器配置
- 确认html目录权限
- 查看生成日志了解错误

### 3. 富文本编辑器无法使用

- 清除浏览器缓存
- 检查网络连接
- 查看浏览器控制台错误

### 4. 图片无法显示

- 检查图片路径是否正确
- 确认图片文件是否存在
- 检查图片文件权限
