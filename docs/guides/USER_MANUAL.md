## 系统介绍

### 功能概览

欢喜内容管理系统（CMS）是一个功能完整、操作简便的内容管理平台，用于创建、编辑、发布和管理网站内容。

#### 主要功能

| 功能模块 | 说明 |
|---------|------|
| **文章管理** | 创建、编辑、发布、删除文章，支持版本控制 |
| **分类管理** | 创建分类体系，支持多层级分类 |
| **标签管理** | 为文章添加标签，便于内容分类 |
| **媒体库** | 上传和管理图片、视频等媒体文件 |
| **用户管理** | 管理系统用户和权限 |
| **系统设置** | 配置网站基本信息、SEO 等 |

### 系统要求

- 现代 Web 浏览器（Chrome、Firefox、Safari、Edge）
- 网络连接
- JavaScript 启用

## 文章管理

### 创建文章

#### 第一步：打开文章编辑页面

1. 点击左侧菜单 "内容管理" > "文章"
2. 点击"新建文章"按钮

#### 第二步：填写基本信息

| 字段 | 说明 | 必需 |
|------|------|------|
| 标题 | 文章标题，建议 5-60 字 | ✓ |
| 分类 | 选择文章所属分类 | ✓ |
| 摘要 | 文章简介，自动生成或手动编辑 | 否 |
| 标签 | 为文章添加标签，便于查找 | 否 |

#### 第三步：编辑内容

使用富文本编辑器编辑文章内容：

- **文本格式**：支持加粗、斜体、下划线等
- **标题**：可设置多个标题等级
- **列表**：支持有序和无序列表
- **链接**：添加内部或外部链接
- **图片**：上传或插入图片
- **代码块**：插入代码片段
- **表格**：创建表格内容

#### 第四步：设置 SEO 信息

1. 展开"SEO 设置"区域
2. 编辑以下信息：
   - **SEO 标题**：搜索引擎显示的标题
   - **SEO 关键词**：3-5 个关键词，用逗号分隔
   - **SEO 描述**：简明的页面描述

#### 第五步：选择操作

| 操作 | 说明 |
|------|------|
| 保存草稿 | 保存为草稿状态，可稍后继续编辑 |
| 提交审核 | 提交给管理员审核 |
| 直接发布 | 直接发布（仅超级管理员） |

### 编辑文章

1. 在文章列表中找到要编辑的文章
2. 点击"编辑"按钮
3. 修改相关内容
4. 点击"更新"按钮

### 发布文章

#### 状态说明

| 状态 | 说明 |
|------|------|
| 草稿 | 尚未发布，只有创建者可见 |
| 待审核 | 已提交，等待管理员审核 |
| 已发布 | 已在前台展示 |
| 已下线 | 已从前台隐藏 |

#### 发布流程

1. 选择未发布的文章
2. 点击"发布"按钮
3. 确认发布信息
4. 文章即刻在前台展示

#### 定时发布

1. 在编辑文章时，展开"发布设置"
2. 启用"定时发布"
3. 选择发布时间
4. 点击保存

系统将在指定时间自动发布文章。

### 文章置顶和推荐

#### 置顶文章

1. 在文章列表中找到要置顶的文章
2. 点击操作菜单中的"置顶"
3. 置顶文章将在列表顶部显示

#### 推荐文章

1. 在文章列表中找到要推荐的文章
2. 点击操作菜单中的"推荐"
3. 推荐文章将在首页特别显示

### 批量操作

#### 批量删除

1. 勾选需要删除的文章复选框
2. 点击顶部"批量删除"按钮
3. 确认删除操作

#### 批量发布

1. 勾选需要发布的草稿
2. 点击顶部"批量发布"按钮

### 文章搜索和筛选

#### 按标题搜索

1. 在文章列表顶部输入搜索关键词
2. 按 Enter 或点击搜索按钮
3. 系统显示匹配的文章

#### 按分类筛选

1. 点击"分类"下拉框
2. 选择目标分类
3. 系统显示该分类下的文章

#### 按状态筛选

1. 点击"状态"下拉框
2. 选择状态（草稿、已发布、待审核等）

#### 高级筛选

1. 点击"高级筛选"按钮
2. 设置多个条件（分类、作者、时间范围等）
3. 点击"搜索"

### 查看文章版本

文章每次保存都会创建一个版本，便于查看历史和回滚。

1. 打开文章编辑页面
2. 点击"版本历史"标签
3. 查看版本列表和对比

#### 对比版本

1. 选择两个版本
2. 点击"对比"按钮
3. 查看两个版本的差异（红色为删除，绿色为新增）

#### 恢复到历史版本

1. 在版本列表中找到要恢复的版本
2. 点击"恢复"按钮
3. 确认恢复

### 删除文章

#### 软删除（移到回收站）

1. 在文章列表中找到要删除的文章
2. 点击"删除"按钮
3. 文章移到回收站

#### 从回收站恢复

1. 点击左侧菜单"回收站"
2. 找到要恢复的文章
3. 点击"恢复"按钮

#### 永久删除

1. 在回收站中找到文章
2. 点击"永久删除"按钮
3. **此操作不可恢复**

## 分类管理

### 创建分类

1. 点击左侧菜单"内容管理" > "分类"
2. 点击"新建分类"按钮
3. 填写以下信息：

| 字段 | 说明 |
|------|------|
| 分类名称 | 分类的显示名称 |
| 上级分类 | 选择父分类，为空则为一级分类 |
| 描述 | 分类的简短描述 |
| 封面图片 | 分类的展示图片 |
| SEO 标题 | 分类页的 SEO 标题 |

4. 点击"保存"

### 编辑分类

1. 在分类列表中找到要编辑的分类
2. 点击分类名称或"编辑"按钮
3. 修改信息
4. 点击"更新"

### 分类排序

1. 在分类列表中，使用鼠标拖拽调整顺序
2. 或者点击"排序"按钮，输入排序号
3. 保存排序

### 删除分类

1. 在分类列表中找到要删除的分类
2. 点击操作菜单中的"删除"
3. 注意：删除分类前，请先移动其下的文章到其他分类

## 媒体库

### 上传文件

#### 单个上传

1. 点击左侧菜单"媒体管理" > "媒体库"
2. 点击"上传文件"按钮
3. 选择文件
4. 等待上传完成

#### 批量上传

1. 点击"上传文件"
2. 选择多个文件（按 Ctrl/Cmd 多选）
3. 系统将逐个上传

#### 拖拽上传

1. 在媒体库空白区域拖拽文件
2. 释放鼠标，文件自动上传

### 支持的文件类型

| 类型 | 格式 | 最大大小 |
|------|------|---------|
| 图片 | JPG, PNG, GIF, WebP | 10 MB |
| 视频 | MP4, WebM | 100 MB |
| 音频 | MP3, WAV | 50 MB |
| 文档 | PDF, DOC, DOCX | 10 MB |

### 查看和管理文件

#### 网格视图

- 查看文件缩略图
- 便于浏览和比较

#### 列表视图

- 查看详细信息（大小、上传时间等）
- 便于搜索和排序

#### 搜索文件

1. 在搜索框输入文件名
2. 系统筛选匹配的文件

### 删除文件

1. 选择文件
2. 点击"删除"按钮
3. 确认删除

## 用户管理

### 查看用户列表

1. 点击左侧菜单"系统管理" > "用户管理"
2. 查看所有系统用户

### 创建新用户

1. 点击"新建用户"按钮
2. 填写以下信息：

| 字段 | 说明 |
|------|------|
| 用户名 | 登录用户名，唯一 |
| 密码 | 初始密码 |
| 真实姓名 | 用户的真实名称 |
| 邮箱 | 联系邮箱 |
| 手机号 | 联系电话 |
| 角色 | 选择用户角色（超级管理员、管理员等） |

3. 点击"保存"

### 编辑用户

1. 在用户列表中点击用户名或"编辑"
2. 修改用户信息
3. 点击"更新"

### 重置密码

1. 在用户列表找到目标用户
2. 点击"重置密码"
3. 设置新密码
4. 通知用户

### 启用/禁用用户

1. 在用户列表找到目标用户
2. 点击"启用"或"禁用"按钮
3. 禁用的用户无法登录

### 删除用户

1. 在用户列表找到要删除的用户
2. 点击"删除"按钮
3. 注意：删除前请转移用户的内容所有权

## 系统设置

### 基本设置

1. 点击左侧菜单"系统管理" > "系统设置"
2. 点击"基本设置"标签
3. 编辑以下内容：

| 设置项 | 说明 |
|--------|------|
| 网站名称 | 前台显示的网站名称 |
| 网站 Logo | 上传网站 Logo |
| 网站描述 | 网站简介 |
| ICP 备案号 | 网站备案号 |
| 版权信息 | 页脚版权文字 |

4. 点击"保存"

### SEO 设置

1. 点击"SEO 设置"标签
2. 配置以下信息：

| 设置项 | 说明 |
|--------|------|
| SEO 标题 | 网站首页 SEO 标题 |
| SEO 关键词 | 网站关键词 |
| SEO 描述 | 网站描述 |
| 自动生成摘要 | 是否自动生成文章摘要 |

3. 点击"保存"

### 上传设置

1. 点击"上传设置"标签
2. 配置上传选项：

| 设置项 | 说明 |
|--------|------|
| 最大上传大小 | 单个文件最大大小（MB） |
| 允许的文件扩展名 | 逗号分隔的文件类型 |
| 存储方式 | 本地或云存储 |

3. 点击"保存"

### 模板设置

1. 点击"模板设置"标签
2. 选择前台使用的模板
3. 点击"保存"

## 常见问题

### Q: 如何恢复误删除的文章？

**A:**
1. 点击左侧菜单"回收站"
2. 找到要恢复的文章
3. 点击"恢复"按钮

文章将恢复到原来的位置。

### Q: 如何为文章设置发布时间？

**A:**
1. 在编辑文章时，找到"发布设置"区域
2. 选择"定时发布"
3. 设置发布日期和时间
4. 保存文章

系统将在指定时间自动发布。

### Q: 如何修改已发布文章？

**A:**
1. 找到要修改的文章
2. 点击"编辑"
3. 修改内容
4. 点击"更新"

注意：修改已发布的文章后，前台将立即显示最新内容。

### Q: 如何查看文章浏览量？

**A:**
在文章列表中查看，或打开文章详情页查看统计信息。

### Q: 支持哪些图片格式？

**A:**
支持 JPG、PNG、GIF、WebP 等格式，最大 10 MB。

### Q: 如何批量导入文章？

**A:**
目前暂不支持直接导入。如需导入大量文章，请联系技术支持。

### Q: 忘记了管理员密码怎么办？

**A:**
1. 联系网站所有者或技术支持
2. 他们可以重置你的密码

### Q: 是否支持文章自动备份？

**A:**
支持。系统每次修改文章都会保存版本。可以在"版本历史"中查看和恢复。

### Q: 如何设置文章预发布预览？

**A:**
1. 编辑文章
2. 点击"预览"按钮
3. 在新窗口查看发布后的效果
4. 确认无误后点击"发布"

### Q: 用户上传的文件存放在哪里？

**A:**
文件默认存放在服务器本地存储，支持配置云存储（如阿里云、腾讯云等）。
